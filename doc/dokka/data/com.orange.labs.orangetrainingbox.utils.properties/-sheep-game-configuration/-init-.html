(JVM) SheepGameConfiguration(moveOffset: Int, walkAnimationPeriod: Long, moveDuration: Long)

Models a bundle of sheep game configuration elements.

Parameters

moveOffset - The move the sheep should make for each rise or fall event, in px

walkAnimationPeriod - Each "frame" of the game icon is animated with this period (in ms)

moveDuration - The duration of each move used by the animator in charge of the sheep icon
