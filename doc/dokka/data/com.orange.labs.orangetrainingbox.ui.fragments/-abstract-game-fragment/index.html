AbstractGameFragment

(JVM) abstract class AbstractGameFragment : AbstractThemedFragment, GameWith3Screens, NavigableGame

Class which models a fragment to use in the navigation graph which embed a game. Factorizes a lot of behaviours about the navigation or the Bluetooth data. Each new game fragment must inherit from this class.
