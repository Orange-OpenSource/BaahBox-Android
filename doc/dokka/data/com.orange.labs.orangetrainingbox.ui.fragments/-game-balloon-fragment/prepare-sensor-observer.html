prepareSensorObserver

(JVM) fun prepareSensorObserver(): Unit

Prepare the Bluetooth LE sensor observer. Defines the behaviour of the observer using the sensor value, and thus the logic of the game. Get from properties the thresholds of the balloon game. Use the value returned by the sensor. Using this value, displays the suitable congratulation message and the image matching the state of the balloon.
