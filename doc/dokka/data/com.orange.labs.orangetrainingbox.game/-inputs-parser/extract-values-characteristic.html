extractValuesCharacteristic

(JVM) fun extractValuesCharacteristic(frame: BluetoothGattCharacteristic?): MuscleData

Using a BLE frame, extracts from it integer values, parse them and return suitable data. Returns a bundle of data with cleaned and computed values for muscles and joystick. If these values are equal to -1, the frame is null.

This implementation follows the frame format of version v2.0.0 of Arduino firmware embedded in the Baah box (see https://github.com/Orange-OpenSource/BaahBox-Arduino/releases).
