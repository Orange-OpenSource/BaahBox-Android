(JVM) CollisionDetector(first: View, second: View, detectionInterval: Long = 500L)

Class which allows to make assertions on objects so as to check if there is a collision including them or not. Computes hitboxes using locations and dimensions, and compare.

Parameters

first - One of the view to use for collision detections

second - One of the view to use for collision detections

detectionInterval - The interval in ms where collision detection is processed (default: 500ms)
