DifficultyFactor

(JVM) enum class DifficultyFactor

Enumeration defining the factor to apply to sensor inputs. The highest the difficulty is, the lowest should be the difficulty coefficient. For example, a DifficultyFactor.LOW implies an high factor.

Parameters

Enum Values

(JVM)

LOW

A low factor to apply, making small the difficulty. Is bound to the 1st value of the seekbar.
(JVM)

MEDIUM

A medium factor to apply, making harder the games thant the LOW level. Is bound to the 2nd value of the seekbar.
(JVM)

HIGH

The highest factor to apply, without help ;-) Is bound to the 3rd value of the seekbar.
