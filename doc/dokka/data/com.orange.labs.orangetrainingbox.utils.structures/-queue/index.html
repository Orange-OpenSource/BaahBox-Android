Queue

(JVM) class Queue<T>

Data structure acting as a FIFO structure. The first element saved in this list will be the first out. Contains a maximum size. If a new element should be added and the queue is full, the first item will be removed.

Parameters

maxSize - The highest number of items this queue can have. If 0, no maximal size is defined.
