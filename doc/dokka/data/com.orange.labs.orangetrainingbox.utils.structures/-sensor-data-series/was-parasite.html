wasParasite

(JVM) private val wasParasite: () -> Boolean

Check in the swap if a parasite value has been stored. If the swap is not empty (no -1 stored), check if the 1st value is far higher (using PARASITE_MAX_FACTOR) than the 2nd. In this case returns true, otherwise false.

Please note that this code works for Arduino firmware v2.0.0 which sends bad values
